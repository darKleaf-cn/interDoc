# 1 注册、登录、用户信息相关

## 1.1 注册

/user/register

post

入参
```json
{
	"username":"",
	"password":"",
	"phone":""
}
```

出参:
```json
{
	"code":200,
	"message":"",
	"result":{
		"userId":1,
		"token":""
	}
}
```

# 2 书籍相关

## 2.1 查询书籍列表

/book/queryList

post

入参

```json
{
	"userId":"",
	//分页相关(必填)

	"page":1,
	"size":20,

	//价格相关(非必填)

	//搜索相关(非必填)

	"name":"明朝那些事",

	//书籍类型(非必填)
	"type":"军事"

}
```

出参

```json

	"code":200,
	"message":"",
	"result":{
		"total":20,
		"data":[
			{
				"bookId":1,
				"name":"",
				"image":"",
				"price":133,

					""
				],

		]
	}
}
```
